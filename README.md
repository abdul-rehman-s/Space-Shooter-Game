# Space Shooter Game (C++)

A simple **console-based Space Shooter Game** developed in **C++** using basic Programming Fundamentals concepts such as loops, arrays, functions, file handling, conditions, and game logic.

## Features

* Player-controlled spaceship
* Enemy waves
* Bullet shooting system
* Collision detection
* Health system
* Score tracking
* Level progression
* Gift/health bonus in Level 2
* High score saving using file handling
* Console animation using `gotoxy()` and `Sleep()`

## Technologies Used

* C++
* Windows Console
* `<conio.h>`
* `<windows.h>`
* `<fstream>`

## Controls

| Key      | Action      |
| -------- | ----------- |
| ↑        | Move Up     |
| ↓        | Move Down   |
| ←        | Move Left   |
| →        | Move Right  |
| Spacebar | Fire Bullet |
| ESC      | Exit Game   |

## Game Levels

### Level 1

* Destroy **10 enemies** to unlock Level 2.

### Level 2

* Enemies move faster.
* Health gifts appear.
* Destroy **12 enemies** to win the game.

## How to Run

1. Open project in **CodeBlocks / Dev C++ / Visual Studio**
2. Compile the `.cpp` file
3. Run the program
